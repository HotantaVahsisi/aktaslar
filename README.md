## Aktaşlar Mühendislik (Static Site)

Bu proje düz HTML/CSS/JS ile hazırlanmış statik bir web sitesidir.

### GitHub'a yükleme
1. GitHub'da yeni bir repo oluşturun (ör. `aktaslar-muhendislik`).
2. Repo içinde **Add file → Upload files** ile bu klasördeki tüm dosyaları yükleyin:
   - `index.html`
   - `hakkimizda.html`
   - `styles.css`
   - `app.js`
   - `assets/` klasörü

### Vercel'e yayınlama
1. Vercel → **Add New → Project**
2. GitHub repo'yu seçin
3. Framework: **Other**
4. Build Command: boş bırakın
5. Output Directory: boş bırakın
6. Deploy
